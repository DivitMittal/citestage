# nix-starter-configs

nix-starter-configs is a beginner friendly NixOS flake template for teams who need reproducible Linux and macOS setups.

## Use cases

Use it to bootstrap Home Manager, share modules, test overlays, and teach new contributors how flakes fit together.

## Quickstart

Run `nix flake init` with the template, choose a host, and follow the guided setup.

## Examples

The examples show laptops, servers, and development shells with clear explanations.
